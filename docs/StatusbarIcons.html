new StatusbarIcons(x, y, icons)

Creates an instance of an icon for the status bar.
Parameters:
Name Type Description
x number The x-coordinate where the icon will be positioned on the canvas.
y number The y-coordinate where the icon will be positioned on the canvas.
icons string The category of the icon which determines the image used. Valid values are 'HEALTH', 'CANDY', 'BOMBS', 'ENDBOSS'.
